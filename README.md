# Rakhine (rki) AI Lexicon

This repository contains a structured, AI‑ready lexical database for the Rakhine (Arakan) language (ISO 639‑3: **rki**).

## Features
- Machine‑readable JSON dictionary entries
- Unified schema for lexicography + NLP
- Corpus folder for monolingual and parallel data
- Scripts for validation and data conversion
- Ready for ASR, MT, embeddings, and LLM fine‑tuning

## Goals
1. Create the most complete digital lexicon of the Rakhine language.
2. Provide open data for AI and linguistic research.
3. Enable community collaboration while keeping data structured and validated.
